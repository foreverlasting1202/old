solution

题面：$n$ 个数，多次询问区间 $[l,r]$，问区间两两或的最小值。

做法：容易观察到最小值一定出自于最小的 $\omega$ 个数中，因为若加入第 $\omega+1$ 个数，若它的 $1$ 在前面都没出现过，则加入它一定更大。否则它一定包含某个数的 $1$，那么也会更大。所以每次只用取出 $\omega$ 个数，直接暴力查一查就行了。时间复杂度 $O(n\omega\log n)$。

[*CF1661 F]

solution

题面：$n$ 个数，要求添加最少的数的个数，然后排序，使得相邻两个数的差值的平方小于等于 $m$。

做法：差分之后，就等价于 $n$ 个数，最少拆分次数，可以拆分数，使得所有数的平方的和小于等于 $m$。

考虑每一个数肯定是想要更加均匀地划分，设 $f(x,k)$ 表示 $x$ 划分成 $k$ 个数时的平方的和的最小值。这个东西是要均匀划分出来的，所以有 $f(x,k)$ 是容易 $O(1)$ 推出的。另一方面，这个东西是下凸的，咋证捏，鼠鼠也不知道，看起来把整除改成实数类比一下是有道理的。

然后考虑对这 $n$ 个凸包贪心，每次肯定会选择差值最大的那个插入，这样复杂度不行。所以二分下斜率，在凸包上再二分出斜率的位置，和 $m$ 比较下即可。时间复杂度 $O(n\log^2 A)$。

坑：二分时我们是将等于二分的斜率的所有差值都选进来了，但其实最终答案可能只用选取一部分，所以二分完之后，我们将大于二分出来的斜率 $+1$ 的选进来，然后把多余的差值除掉即可。

[CF1667 B]

solution

题面：$n$ 个数，划分成若干段，每一段的贡献为这一段的数的和的正负乘上区间长度，问最大总贡献。

做法：设 $f_i$ 表示 $i$ 处截止的最大总贡献，转移就是 $f_i=\max\lbrace f_j+(i-j)\cdot sign(s_i-s_j)\rbrace$，其中 $s_i$ 表示前缀和。按 $sign$ 分类讨论，两类用线段树维护即可。离散化下节省空间。时间复杂度 $O(n\log n)$。

[*CF1667 C]

solution

题面：只能影响一边对角线的皇后，$n\cdot n$ 的网格图，问最少多少个皇后使得每个格子都被控制。

做法：这种题应该先去找到下界，可惜考场上没想到。

假设答案为 $m$，那么至少有 $n-m$ 行和 $n-m$ 列没有被直接控制，设这些行列为 $r_1,r_2,\cdots,r_a,c_1,\cdots c_b$，那么对于这些行列所形成的最靠左上的直角形而言，每个点都不在同一条对角线上，即 $(r_1,c_1),(r_1,c_2),\cdots,(r_1,c_b),(r_2,c_1),\cdots,(r_a,c_1)$。所以这些点肯定要被 $m$ 个皇后的对角线控制住，而每个皇后只能控制这其中的一个。所以有 $a+b-1\leq m$，以及 $n-m\leq a,n-m\leq b$。化简，有 $m\geq (2n-1)/3$，于是就得到了个下界。

接下来想办法取到这个下界。先考虑整除的情况，即 $n=3x+2$，此时 $m=2x+1$ 且 $a=b=x+1$。于是根据前面的证明，不妨让扔 $x+1$ 到左上，$x+1$ 到右下等等。反正方法一大堆。不整除随便添加两个点即可。

solution

题面：给定一张无向图，求出与 $(s,t)$ 之间的与最短路长度差不超过 $1$ 的路径条数。

做法：正着跑个最短路，倒着跑个最短路，顺便算出最短路的路径数。然后枚举一条边作为次短路的边，两个端点乘起来看起来是答案，但实际算重了好多。为了不算重，注意到一条最短路上的某个点连出一条不在最短路上的边这样算就是唯一的了。于是就做完了。时间复杂度 $O(n+m)$。

solution

题面：长度为 $n$ 的序列 $a$，现给出一个 $a$ 的排列 $b$，你只有一种操作：互换两个数的位置。问 $b$ 变到 $a$ 的操作次数是不是所有 $a$ 的排列变到 $a$ 的最大操作次数，顺便询问最大操作次数的序列。

做法：对于这种题，都是考虑交换序列来连边，然后分析这张图的，这题也不例外。

对于一张已经连好的图而言，由于它是一个个环组成的，所以答案就是 $n-$ 环的个数。然后一个很显然的结论，对于任意一个排列 $b$ 而言，每个环上都会包含且只包含一个出现次数最多的数 $x$。这个结论是容易猜的，因为你建出那张完整的图的时候，想划分环时一定是这样贪心的。证明也容易，考虑反证法，即假设一个环有两个 $x$，不妨设 $b_{i_1}=b_{i_t}=x$，然后环是 $i_1,i_2,\cdots,i_t,\cdots,i_m,i_1$。此时显然可以拆成两个环，即 $i_t,i_2,\cdots,i_{t-1},i_t$ 和 $i_{1},i_{t+1},\cdots,i_m,i_1$。所以就矛盾了，于是得证。

有了这个结论就做完了，构造最大的序列直接就令出现最多的数为开头，强行连环即可。而判断是否为最大稍微麻烦，因为此刻我们并不知道这张图的真实模样，直接 $a_i\rightarrow b_i$ 连边的图还有一些无用边。但因为这张图最多环时要每个 $x$ 掌管一个环，所以将 $x$ 去掉时，这张图要是张 DAG （ 因为如果不是则可以让剩余的那些点构成一个环，那就不是被 $x$ 掌控了）。所以 dfs 判下 DAG 即可。

solution

题面：$n\cdot n$ 的网格图，每个位置上有一个不同的数，当上个人走到某个位置时，下个人只能走到与上个人的曼哈顿距离大于 $k$ 的地方。当一个人走到某个位置时，可以获得这个位置上的数，每个位置的数可以无限次获得。问第一个人从 $n\cdot n$ 个位置出发时，无限轮后，是第一个人分数多还是第二个人分数多。

做法：首先，谁先站到最大值上谁就赢了。于是初始位置在最大值所包揽的正方形外就输了，否则不能离开那个正方形。若初始在那个最大值正方形内的次大值上，则不管对方如何移动，你都可以回到这个次小值而战胜对方。若在最大值正方形和次大值正方形的差集上，则先手已经输了。否则就要在最大值正方形和次大值正方形的交集里。若初始在这个交集的第三大值上，则已经必胜，否则跟上述一致。于是我们一直去缩小这个交集，从而能得到所有的先手必胜点。

实现上，考虑从最大的往小的找，然后缓慢缩小正方形边界，当当前点已经不在正方形里就寄，或者继续缩小。时间复杂度 $O(n^2)$。

关于如何维护那个交集，等价于维护四条边界直线即可，于是根据直线形式将整个坐标轴翻转一下，维护就变得异常简单了。

solution

题面：交互，未知一个数组 $l$，每次询问一个 $s$，划分 $l$ 使得每一段的和加上这一段长度减一小于等于 $s$，返回的是最小划分数。$n+30$ 次内猜出最小的 $s\cdot x$，$x$ 表示对应的最小划分数。

做法：不算特别难的一题。

说下我的思路历程，首先能观察到的时候最优划分一定是从左往右嗯贪心的。现在我想要枚举一共被划分成多少个区间，又想立刻知道划分成 $k$ 个区间时最小的 $s$，发现这件事情不太容易处理。假设划分 $k$ 个时的区间和加长度减一为 $A_1,\cdots,A_k$ ，那么最小的 $s$ 就是 $\max\lbrace A_i\rbrace$。好像并不好处理。我们注意到题目有个重要性质，它的划分并不是区间和，而是再加上长度减一。这意味着不同的 $k$ 就有不同的 $\sum A_i$。设 $n-1+\sum l_i=S$，那么对于一个 $k$ 而言，此刻的 $\sum A_i=S-k+1$。有趣的事情发生了，注意到询问 $S$ 时答案是 $S$。那么若想获得更小的答案，则那个答案一定小于 $S$ 了。而对于 $k$ 而言，它的答案是 $k\cdot \max\lbrace A_i\rbrace$，这个东西是大于 $\sum A_i=S-k+1$ 的，但我们却要它小于 $S$。这意味着 $k\cdot\max\lbrace A_i\rbrace<S$，也即 $\max\lbrace A_i\rbrace<\frac{S}{k}$。而另一方面，由于 $\sum A_i=S-k+1$ 的，所以 $\frac{S-k+1}{k}\leq\max\lbrace A_i\rbrace$。这样的上下界刚好把 $\max\lbrace A_i\rbrace$ 卡死，所以只用 $n$ 次询问 $\frac{S}{k}$ 即可，以及开始 $20$ 多次二分询问出 $S$。

[*CF549 E]

solution

题面：给定平面上两组整点，分别有 $n,m$ 个，坐标范围在 $[-R,R]$ 之间，问是否存在一个圆，满足其中一组点均在圆内 (含边界)，另外一组点均在圆外 (不含边界)。

做法：先考虑一个做法，我们强行找到每个包含一组点的圆（显然经过两点更不劣），判断另一组点是否在其外。这样显然存在一个问题，我们不太好枚举所有包含一组点的圆，直接暴力复杂度不符合要求。

回忆起 [ZJOI2018] 保镖，我们将平面一个点 $(x,y)$ 映射到 $z=x^2+y^2$ 的抛物面后，对一组点求出三维凸包，我们可以证明上凸包的平面与支配圆一一映射（定义支配圆为包含一组点的圆）。这个性质的证明如下：

我们先写出平面的圆方程: $x^2+y^2+Dx+Ey+F=z+Dx+Ey+F=0$，将 $(x,y,z)$ 看作空间里的一个点后，圆方程就可以被看成一个平面。因此，一个点在圆内外等价于这个点的映射在平面的哪一侧。

于是，一个支配圆意味着所有的点的映射都在这个支配圆的映射平面的内侧。因此，求出三维凸包后，上凸包的内侧就是指在上凸包里面。也即上凸包的每一个平面都对应一个支配圆，每一个支配圆都对应一个上凸包上的平面。

为了确保唯一性，我们对所有点做一个微小扰动，使得平面上没有四个点是共圆的，保证了唯一性。

这样我们得到了一个比较优秀的做法，求出三维凸包后，枚举上凸包的每一个平面，看看是否隔断两组点。但由于三维凸包的 $O(n\log n)$ 做法比较复杂，我们重新想回二维。

我们重新将上凸包投射到二维空间里，我们可以称其为支配三角剖分或 Anti-Delaunay Triagulation。这个三角剖分满足每个三角形的外接圆包含一组点，而且我们知道这样的三角形个数是 $O(convex(n))$ 级别的。（$convex(n)$ 表示凸包上的点数）

现在的关键就是求出这个三角剖分，我们考虑分治。开始先枚举一条边，然后找到唯一确定的支配三角形（唯一性上面已经证明），然后按着三角形的剩下两条边切开多边形，继续分治下去。由于它映射到空间的三维凸包是惟一的，所以这样的分治也是惟一且正确的。

于是总复杂度就是 $O(convex(n)(n+m)+n\log n+m\log m)$。

而 $O(convex(n))=O(R^\frac{2}{3})$，于是我们就做完了这道题。

具体实现上学习了鱼大的做法，计算弧度来避免使用开方等操作而减少了精度。

题面：给定一个长度为 $n$ 的序列和一个整数 $p$。有 $m$ 个操作，操作要么是区间赋值，要么是询问区间内出现次数至少占 $p\%$ 的数。输出询问的答案时，可以包含错的数，也可以重复输出，但对的数一定要在答案中，且输出的数的个数不超过 $\lfloor\frac{100}{p}\rfloor$。

做法：考虑摩根（ 忘记名字了 ）投票法，就那个加一个就删一个的。注意到 $p\geq 20$，所以最多维护五个数。所以线段树每个节点维护个五元组就好了，时间复杂度 $O(5^2n\log n)$。

[*CF594 E]

solution

题面：给定一个字符串 $s$ 和一个正整数 $k$，你可以将 $s$ 分成至多 $k$ 段，并将每一段翻转或者不翻转。求最终能够得到的字典序最小的 $s$。

做法：如果是必须翻转，这题将会是道不太难的题，因为等价于反串后缀排序后不断取最小值就行了，但因为有不翻转，导致不翻转的起点位置和终点位置对数变成 $O(n^2)$，所以我就不会了。

然后看了下题解，才想起来这题是 $2020$ 集训队作业里过题人数最少的一题，趁机学了这一系列的字符串算法。

先定义 $\mathrm{Lyndon}$ 串，一个串被称为 $\mathrm{Lyndon}$ 串当且仅当这个串的最小后缀是它本身。

可以容易证明这个定义等价于这个串是它的所有循环表示中字典序最小的。

然后可以证明一个小引理。

引理一：若 $u,v$ 是 $\text{Lyndon}$ 串，且 $u<v$，那么 $uv$ 也是 $\text{Lyndon}$ 串。

证明：反证，假设不是，则存在一个后缀 $w$，使得 $w<uv$。讨论一下 $w$ 的情况。

第一类：$|w|\leq |v|$。这意味着 $w$ 是 $v$ 的一个后缀，但 $u<v$ 且 $v$ 是 $\text{Lyndon}$ 串，矛盾。

第二类：$|w|>|v|$。这意味着 $w[1,\cdots,p]$ 是 $u$ 的一个后缀，这与 $u$ 是 $\text{Lyndon}$ 串矛盾。

于是得出 $uv$ 是 $\text{Lyndon}$ 串。

根据这个引理，我们有一个对字符串的划分手段，叫 $\text{Lyndon}$ 分解。下面给出定义：

我们定义一个串 $S$ 的 $\text{Lyndon}$ 划分为一个字符串序列 $A_1,A_2,\dots,A_m$，满足：

  1. $S=A_1A_2\cdots A_m$。
  2. 对任意的 $i$，$A_i$ 是 $\mathrm{Lyndon}$ 串。
  3. $A_i\geq A_{i+1}$。

可以证明这个划分一定存在且唯一。

存在性是显然的，考虑每次将相邻的两个满足 $A_i<A_{i+1}$ 的串合并，可以显然证明此时每个串是 $\mathrm{Lyndon}$ 串，同时满足条件三了。

唯一性证明仍然用的反证法，假设存在两种划分手段 $S=A_1A_2\cdots A_iA_{i+1}\cdots A_m=A_1A_2\cdots A’_{i}A’_{i+1}\cdots A’_n$。考虑 $A_i$ 和 $A’_{i}$，不妨设后者是前者的前缀，那么 $A_i=A’_iA’_{i+1}\cdots A’_kA’_{k+1}[1,\cdots ,l]$，于是要有 $A_i<A’_{k+1}[1,\cdots,l]\leq A’_{k+1}\leq A’_i<A_i$，矛盾。所以唯一性得证。

再做一个铺垫，继续证一个引理。

引理二：若字符串 $u$ 和字符 $c$ 满足 $uc$ 是某个 $\text{Lyndon}$ 串的前缀，则对于 $d>c$ 有 $ud$ 是 $\text{Lyndon}$ 串。

证明：设某个 $\text{Lyndon}$ 串为 $uct$，那么有对任意的 $i$，满足 $u[i,\cdots,n]ct>uct$，这意味着 $u[i,\cdots,n]c\geq u$，于是 $u[i,\cdots ,n]d>u[i,\cdots,n]c\geq u$，而 $ct>uct$ 可以知道 $c\geq u[1]$，那么 $d>c\geq u[1]$，于是 $ud$ 是 $\text{Lyndon}$ 串。

有这两个引理后我们就可以尝试在 $O(n)$ 的时间内做到对一个串的 $\text{Lyndon}$ 分解了，一般称这个算法为 $\text{Duval}$ 算法。

solution

题面：给一棵树，每条边有边权，从 $1$ 开始随机游走，问到达任意一个叶子的期望时间。

做法：考虑 $f_x$ 表示 $x$ 开始游走的期望时间，有转移 $f_x=\frac{f_{fa}+w_x+\sum f_{to}+w_{to}}{du_x}$，这东西带环，不过归纳法容易证明 $f_x$ 是 $f_{fa}$ 的一次函数，所以可以手动高斯消元出一次函数表达式，初始态是叶子时等于 $0$。手动高消过程这里就不展现，具体看代码。时间复杂度 $O(n)$。

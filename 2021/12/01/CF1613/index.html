哈哈，最后一小时才发现第二天我生日，然后就去水群了，最后两题没写，赛后看一下还蛮简单的。

[A Long Comparison]

solution

题面：给出 $x_1,p_1,x_2,p_2$，问 $x_1\cdot 10^{p_1}$ 与 $x_2\cdot 10^{p_2}$ 的大小关系。

做法：除一下就行了。

[B Absent Remainder]

solution

题面：给出 $n$ 个互不相同的数 $a_i$，需要你找出 $\lfloor\frac{n}{2}\rfloor$ 组 $(x,y)$，使得 $x,y$ 都在这 $n$ 个数且 $x\mod y$ 不在这 $n$ 个数里。

做法：我还以为是匹配问题呢，想了半天不会。一看样例发现可以重复，那写啥 $\lfloor\frac{n}{2}\rfloor$ 啊。直接 $y$ 是最小的数就行了。

[C Poisoned Dagger]

solution

题面：给出 $n$ 个不同的数，找到最小的 $k$，使得所有减去这 $n$ 个数小于等于 $k-1$ 大于等于 $0$ 的数的个数大于等于 $h$。

做法：二分，结束。时间复杂度 $O(n\log h)$。

[D MEX Sequences]

solution

题面：给出 $n$ 个数，问满足前缀 $\mathrm{mex}$ 与前缀的最后一个数的差值绝对值小于等于 $1$ 的子序列个数。

做法：注意到满足条件的序列的性质，可以发现它要么是 $0,1,…,p$ 的连续序列，要么是 $0,1,…,p,p+2$，即中间跨越一个 $p+1$。前者每次加入一个新的数到最后，只能是 $p$ 或 $p+1$ 或 $p+2$，后者只能是 $p,p+2$。根据这个性质可以设计一个 $\mathrm{dp}$，设 $f_{i,j,0/1}$ 表示当前 $\mathrm{dp}$ 到第 $i$ 个数，以第 $i$ 个数为结尾的序列是连续型还是跨越一个型，以及 $p=j$。转移分几种，但大体研究下，可以发现第二维无意义，只需要再设 $g_j$ 表示 $p=j$ 的连续序列的方案数，$h_j$ 表示 $p=j$ 的跨越一个型序列的方案数，$f_{i,0/1/2}$ 表示以第 $i$ 个数为结尾，是连续序列/跨越一个型但 $p=a_i$/跨越一个型但 $p=a_{i}-2$ 的方案数。发现这样转移就很简单了。时间复杂度 $O(n)$。

[E Crazy Robot]

solution

题面：$n\cdot m$ 的网格图，有些位置是障碍，有个位置是终点。你可以让机器人以任意一个位置作为起点，每次你可以给出一个向四个方向走的指令，但机器人一定不会执行你的指令，同时它不会撞墙。问哪些起点走它可以走到终点。

做法：考虑倒回来走，当只有 $1$ 种或 $2$ 种选择的时候它才会听话。所以直接倒着 $\mathrm{bfs}$ 即可。时间复杂度 $O(n\cdot m)$。

[F Tree Coloring]

solution

题面：一棵节点数为 $n$ 的树，然后每个节点分别染 $1\backsim n$ 的颜色，且每种颜色用且只用一遍。问父亲颜色不是儿子颜色 $+1$ 的染色方案数。

做法：大裸题。

注意到颜色的排列对应点的序号的排列，所以可以考虑容斥。设 $f_i$ 表示选取了 $i$ 条边使得这 $i$ 条边连接的父子不满足条件，注意到颜色也是排列，所以要求这 $i$ 条边不交于同一个父亲。这样的话，就等价于对每一个父亲，有它儿子数量种选择方案。容易写出 $f$ 的封闭形式，即 $f_i=[x^i]\prod (1+du_j x)$，其中 $du_j$ 表示 $j$ 这个节点的出度。这东西直接分治 $\mathrm{FFT}$ 一下解决。于是答案就是考虑容斥，即 $\sum_i (-1)^i (n-i)!f_i$。总复杂度 $O(n\log^2n)$。（ 如果真有人喜欢多项式 $\exp$ 的话也可以 $O(n\log n)$ ）

[1001 Yes, Prime Minister]

solution

题面：给出 $x$，求 $(l,r)$ 使得 $l\leq x\leq r$ 且 $\sum_{i=l}^r i$ 为正质数且该正质数最小。$l,r$ 可以为负数。​​

做法：先求个和，即 $\frac{(r-l+1)(l+r)}{2}$​​。这个要为正质数，至少要求 $l+r=2$ 或 $l+r=1$ 或 $r-l+1=1$ 或 $r-l+1=2$​​，后面两类可以手玩，前两类手玩后等价于找一个质数。可以二分，也可以直接暴力找。反正都能过。

时间复杂度 $O(\log)$ 或 $O(质数距离)$。

[1002 Might and Magic]

solution

题面：两个人，每个人都有五个属性：$A_i,D_i,P_i,K_i,H_i$​。还有两个参数 $C_p,C_m$​。第 $i$​ 个人对第 $1-i$​​ 个人物理攻击伤害为 $C_p\max(1,A_i-D_{1-i})$​，魔法伤害为 $C_mP_i$​。$K_i$​ 表示第 $i$​ 个人可以使用魔法攻击的次数。$H_i$​ 是第 $i$​ 个人的生命值。$P_1=K_1=0$​，现给出 $N$​，你可以将数值分配到 $A_0,D_0,P_0,K_0$​ 上，$C_p,C_m,A_1,D_1,H_0$ 已知。问最多可以打掉第二个人多少生命？​

做法：若分配之后物伤大于等于法伤，那我肯定全选择物伤，则让关于法伤的都为 $0$​。否则当法伤大于物理伤害时，我们枚举总攻击次数 $t$​ 和 $K_0$​，把 $A_0$​ 分配成 $0$​​​，算出此时的值，然后与 $A_0\neq 0$​ 做比较，会发现此时的值一定大于其他值（敲公式太麻烦了，懒得敲了），也即最大值情况会在只分配物伤或只分配法伤时取到。分配物伤是直接的，分配法伤是个二次函数，取对称轴两边即可。而我们知道 $t=\lceil\frac{H_0}{C_p\max(1,A_1-D_0)}\rceil$​ 只有 $O(\sqrt{H_0})$​​​ 种​，所以枚举 $t$ 后总复杂度就是 $O(\sqrt{H_0})$​​ 了。

官方题解是注意物伤关于 $A_0$ 是下凸函数，法伤关于 $P_0$​ 也是下凸函数，下凸函数的和仍然是下凸函数，所以答案一定在端点处取到。

坑：向上取整我们可以分子减一再整体加一变成向下取整。

[1003 0 tree]

solution

题面：一棵树，有点权 $a_i$​​​​ 和边权 $b_{i}$​​​​，每次可以选择一条从 $x$​​​​ 到 $y$​​​​ 的路径 $(v_0,v_1,…,v_k)$​ 和权值 $w$​，其中 $v_0=x,v_k=y$​，使 $a_x\leftarrow a_x\oplus w,a_y\leftarrow a_y\oplus w,b_{e_i}\leftarrow b_{e_i}+(-1)^iw$​，$e_i=(v_i,v_{i+1})$​。现让你构造不超过 $4n$ 次操作使得所有点权和边权变为 $0$。

做法：先将边权转为点权，即每个点的第二点权为连接它的边的权值和。那每次操作只与端点有关了。

注意到几件事，首先每次操作的总异或和不变，所以总异或和为 $0$​​ 是有解的充分条件。其次，每个点第一权值和第二权值的奇偶关系是始终不变的，所以奇偶相同也是一个充分条件。另一方面，我们发现加减的不同仅在两个点的相隔点数的奇偶，于是考虑黑白染色，即黑点与白点相邻。那么只有异色才是都加，同色是一加一减。那么同色点间的操作是不会改变这种颜色的第一与第二权值和，能改变只能来自异色操作。而 $x+y\geq x\oplus y$，所以每种颜色的第一权值异或和小于等于第二权值和的负数也是一个充分条件。实际上，以上已经构成充要条件。接下来我们来构造方案。

考虑第一权值和第二权值分开来处理。

我们发现同色间进行 $(x,y,v),(x,y,v),(y,x,2v)$​ 会使

$a_x\leftarrow a_x\oplus(2v),a_y\leftarrow a_y\oplus(2v)$。

同色间进行 $(x,y,v),(x,y,v)$​ 会使

$b_x\leftarrow b_x+2v,b_y\leftarrow b_y-2v$。

于是若同色层的第一权值异或和为 $0$​ 且每个点第一权值为偶数的话，按照第一种操作我们就可以让每个点的第一权值为 $0$​，那么就处理好第一权值了。若同色层的第二权值和为 $0$​​ 且每个点第二权值为偶数的话，按第二种操作我们就能处理好第二权值了。

于是问题变成如何让同色层的异或和以及权值和都为 $0$​​​ 且每个点的点权都是偶数呢？假设黑色异或和以及权值和分别为 $A,B$​​​，白色的分别为 $C,D$​​​​。我们根据充分条件，暂时得到的是 $A=C,A\leq -B,C\leq -D$​​​，以及 $A$​ 和 $B$​ 同奇偶，$C$​ 和 $D$​​ 同奇偶。我们选择两个异色点，执行 $A,-\frac{A+B}{2},-\frac{A+B}{2}$​ 三次操作，发现这样 $A’=B’=C’=0,D’=D-B$​​。好像 $D’$ 不符合我们预期？再冷静一下，一条边转成点权的时候，一部分是变成黑色，一部分是变成白色，于是 $B=D$，因此我们成功做到了权值和为 $0$​。那全是偶数的条件该怎么办呢？注意到每个点的两个权值的奇偶性是相同的，我们先对第二权值动刀，即每个点减去自己的第二权值。这样可以使每个点的第二权值变为 $0$（因为总权值和为 $0$​，而这种操作不会改变总权值和），同时由于 $0$ 是偶数，所以不知不觉第一权值也都是偶数了。于是就成功了。

计算一下操作次数，是 $3+n-2+3(n-2)=4n-5$ 次，刚好通过。

另一方面，先处理第一权值的话，操作次数更少，只有 $3+n-2+2(n-2)=3n-3$。

坑：由于 c++ 异或负数会出错，所以要每一次操作要注意一下。

[1004 Decomposition]

solution

题面：一张 $n$​ 个点的无向完全图，你需要找出 $k$ 条无重复点的路径使每条路径长度分别等于给出的长度 $l_i$，保证 $\sum l_i$ 为总边数且 $l_i\leq n-3$。

做法：直接构造出一组解，别问我是怎么构造的。设 $n=2m+1$。

$(2m+1,1,2m,2,2m-1,…,m,m+1),(2m+1,2,1,3,2m,…,m+1,m+2),…$​

感觉这种比 $n-3$ 要强？​​

[1005 Median]

solution

题面：给出 $b_i$，现希望你将 $n$ 的排列分成 $m$ 组，使得每一组的中位数为 $b_i$​。

做法：先分析下中位数条件。假设一组里大于中位数的个数是 $R$，小于是 $L$，则 $L\leq R\leq L+1$。

然后再来思考限制中位数意味着什么。我们将 $1\backsim n$ 的排列用 $b_i$​​ 隔开，分成 $m+1$ 组别（空的也算）。那么选择两个数加入同一组里就等价于选择不同组别的数。注意到 $R\leq L+1$，这意味着越后面的组别就有更大的机动性，它们可以插入到前面那些组里。即第 $i$ 组别的数的个数至少要小于等于 $i-1$。于是我们就设计了一个贪心思路。我们把所有大于 $i-1$ 的部分拿出来互相消除，这样除非有一组的个数大于其他组的和，不然就只剩 $1$ 个或 $0$​ 个（考虑总数的奇偶）。剩 $0$ 个肯定有解了，剩 $1$ 个我们希望把这个 $1$ 放在最后面（这是可以做到的），然后就看前面不大于 $i-1$ 的部分有没有可以消除的。剩余多个也是这个道理。

如果还没看懂就看代码吧，感觉有点难以讲明白。

复杂度 $O(n)$​ 或 $O(n\log n)$，瓶颈在分成 $m+1$ 组别的过程。

[1006 The Struggle]

solution

题面：给出 $a,b,c,d,e,f$，$E =\{(x,y)| x,y \in \mathbb{Z},a(x-b)^2+c(y-d)^2+e(x-b)(y-d) \le f\}$。

求 $\sum_{(x,y) \in E} (x \oplus y)^3x^{-2}y^{-1} \mod 10^9+7$。​

做法：容易考虑枚举 $x\oplus y$，那后面的东西就是个卷积的东西，$\mathrm{FWT}$ 即可。

但是呢，每一个 $x$ 对应的 $y$ 都是不同的，那直接全部卷起来肯定是不太行的。

[1007 Power Station of Art]

solution

题面：给你一个有标号无向图，每个点有黑或白的颜色以及它自己的标号，每次可以选一条边并交换上面两个点的标号，但是如果两个点的颜色相同，这两个点的颜色会同时改变。现在给你两个形态一致的图，问第一张图经过操作之后能否变为第二张图。

做法：注意到交换操作的等价形式是交换标号和颜色，再反转颜色。于是我们的交换就可以去掉 if 条件了。

那么对于一个标号而言，交换次数的奇偶就代表了它的颜色。这让我们想了二分图。

如果是一张二分图，一个标号在哪一边就意味着它的颜色了。所以只要标号数量和颜色是对应的，那我们一个个按类似拓扑序的顺序来交换点，就保证能构造出方案了。

如果不是二分图，那每个标号的颜色就可以任意换了。于是我们只要标号数量是对应的，利用奇环让每个标号的颜色对应，就可以转成二分图时的构造了。

时间复杂度 $O(n+m)$。

坑：1. 图未必连通，要一个个连通块处理。

​ 2. 标号不唯一，可能重复，所以要按数量算。

[1008 Command and Conquer: Red Alert 2]

solution

题面：给出三维空间里若干个点，你从任意点出发，每次只能向坐标递增的方向走，路径由你自己构造。问你走的路径到这若干个点的切比雪夫距离的最小值。

做法：考虑二分答案。那么那就等于在一堆正方体中找出一条路径满足每个正方体都被碰到。一个贪心就是找到一个 $x/y/z$​​ 坐标最小的正方体，然后走到这个最小的 $x/y/z$​​ 坐标。这显然是最优的，因为不走白不走，一定是不劣的。于是我们已经得到了 $O(n\log n)$ 的做法。

再冷静一下，发现也没必要二分答案了。考虑直接将所有点平移到正方体的左前下角，那么我们每次贪心想找的点就是这些点本身了，直接找就行了。时间复杂度也是 $O(n\log n)$​​​，瓶颈在排序。

坑：实现下面那个做法的时候，注意到所有点平移了，所以这里求得的 $k$ 是平移后的 $k$。而正方体中心平移到一个端点是边长乘 $2$，那么 $k\leftarrow \lceil\frac{k}{2}\rceil$ 即可。

[1009 Typing Contest]

solution

题面：每个人有两个数值 $s_i,f_i$，现让你挑出若干个人，使得 $\sum s_i(1-f_i(-f_i+\sum f_j))$ 最大。​

做法：稍微做一些化简，即

发现前面两个东西和 $\sum f_i$​​​ 无关，只有最后一个东西有关。考虑枚举 $F=\sum f_i$​​，容易想到背包，即设 $g_{i}$​​ 表示当前 $\sum f_j=i$​​​​ 的最大价值，那么求的就是 $g_F$​。注意到 $f_i$​ 是两位小数，那么直接枚举的复杂度就是 $O((100fn)^2n)$​，轻而易举地过不去。

然后考场上 ljc 迅速发现 $\sum f_i$ 实际上不用枚举那么多，而且是容易证明的。

我们还是考虑对每一个人进行计算，那么对一个人而言的贡献就是 $(-F\cdot f_i+1+f_i^2)s_i$。贪心地想，肯定要系数大于 $0$，即 $1+f_i^2>f_i\cdot F$。把 $f_i$ 直接放到 $1$，那么 $f_i\cdot F<2$。假设我们选了 $k$ 个人，两边同乘 $k$​，则 $kf_i\cdot F<2k$。而必然存在一个人的 $f_i$ 是最大的，所以 $kf_i\geq F$，也即 $F^2<2k\leq 2n$，于是 $F<\sqrt{2n}$。

这样的话，复杂度就变成了 $O((100\sqrt{2n})^2n)=O(4\ast10^4n^3)$​，这大概是可以跑过去的，不行就再剪剪枝，毕竟上面的放缩是比较松的。​

坑：1. 注意 $f_i\leftarrow f_i\times 100$​ 后，$1$​ 也要乘 $10000$​​​。​

​ 2. 虽然他题面是让你输出九位小数，但他让你输出的小数是在 $f_i$ 还是两位小数的时候，所以最后五位一定是 $0$ 。如果直接除以 $100000$​​ 会产生精度损失，所以建议取模来拿到小数部分。

​ 3. 乘上 $100$ 时注意是有精度损失的，浮点数存到 $16$ 位，那损失的量大致是 $2^{-15}\times 100$，所以要加上 $0.04$ 差不多避免精度损失。​​

[1010 Array]

solution

题面：给出一个 $B$​ 数组，满足 $B$​ 数组单调不减。设 $S(T)$​ 表示有多少个不同的数在 $T$​ 集合里，询问是否存在一个数组 $A$​，使得对任意的 $l,r$​，$S(A[l,r])=S(A[1,n])$​ 是 $r\geq B_l$​​ ​的充要条件？

做法：我们刚开始肯定是想直接构造 $A$ 数组，可惜想了一圈之后头皮发麻，只感觉限制之间非常难以处理。而比较好的做法则是构造 $b$ 数组，其中 $b_i$ 表示最前的在 $i$ 之后与 $i$ 颜色一致的 $j$，即 $\min_{i<j,a_i=a_j} j$，为了方便，不存在 $j$ 则设为 $n+1$。如果我们构造出了一个 $b$ 数组，则很容易就能构造出 $A$ 数组。事实上，这个 $b$ 数组要比 $A$ 数组容易构造许多。

我们考虑 $b$ 与 $B$ 的关系。经过分类讨论，我们发现 $B_{i+1}=\max(B_i,b_i)$，这是相当容易证明的。而且通过证明它的过程，我们发现，满足这个条件的 $b$ ，一定就是合法的 $b$，也一定能构造出 $A$。而利用这个条件就直观许多。若 $B_{i+1}>B_i$，则 $b_i=B_{i+1}$。若 $B_{i+1}=B_i$，则 $B_{i}\geq b_i>i$。但仅有这样的不等式是不容易确定 $b_i$ 间的矛盾的，我们细化这个构造的过程。

首先，$[B_1+1,n]$ 的 $A$ 一定都不是第一次出现的，这可以由 $[1,B_1]$ 这个区间限制来证明，而无法证明的情况则是 $B_1=n+1$，那就特判掉。一个位置的数不是第一次出现这个条件转到 $b$ 上，就是存在一个 $b_i=j$，且 $j$ 是这个位置。于是，$[B_1+1,n]$ 是构造 $b_i$ 使必选的，剩余的数包括 $n+1$ 是可选可不选的，且所有数（除了 $n+1$）都只能选一次。

接下来我们看那个不等式的限制，即 $i<b_i\leq B_i$，我们发现这个不等式的左端点与右端点都是单调的。这给我们一定的贪心启示。我们设必选数的集合为 $A$，其余为 $B$，每选出一个就将其删去（除了 $n+1$）。$i$ 从小到大扫过去，设当前 $A$ 中最小值为 $x$，$B$ 中最小的大于 $i$ 的为 $y$，接下来开始基于贪心着讨论。

  1. $x\leq i$。这一定不合法了，因为无处放 $x$。
  2. $x>B_i,y>B_i$。这也一定不合法了，因为 $b_i$ 不合法。
  3. $x$ 和 $y$ 只有一个不合法。那就只能选合法的那个了。
  4. $x$ 和 $y$ 都合法，但 $x<y$。注意到扫描到后面时，若可以放 $x$ 就一定可以放 $y$，所以我们选择更不劣的方案，这里放 $x$。
  5. $x$ 和 $y$ 都合法，但 $x>y$。我们暂时的方案是放 $y$，但同时从 $A$ 中去掉 $x$，在 $B$ 中加入 $x$。若最终构造方案有了 $x$，那这种方案无问题。若没有，我们将 $b_i$ 变成 $x$ 即可。而且，这种暂时的方案是不会影响到其他构造的，理由同上。

于是，我们就成功做到了最优的构造。实现上，维护这两个集合可以用 set 或者 list，做到时间复杂度 $O(n\log n)$ 或 $O(n)$。为了方便，我写的是 $O(n\log n)$。

[1011 Game]

solution

题面：两个人在博弈，设当前区间为 $[l,r]$，两个人的操作为使区间变成 $[l-1,r]$ 或 $[l,r-1]$，一个人面对的区间是 $[x,x]$ 时输，再给出若干个区间 $[a_i,b_i]$ 以及对应的参数 $c_i=0/1$，表示当一个人面对的区间是 $[a_i,b_i]$ 时这个人直接判胜或判负。若干组询问，每次询问一个初始区间 $[p,q]$，问谁胜？

做法：说下我考场的思路。我们先将状态转移图写出来，发现如果没有那些特殊区间，$[l,r]$​​ 与 $[l-1,r+1]$​​ 的情况是一致。也就是胜负取决于 $l+r$​​ 的奇偶。如果有那些特殊区间应该怎么办呢？我们会发现只要离那些特殊区间两步之远的都不会受影响。这是容易证明的（$0$​ 表示先手必负态）。若 $[l,r]$​ 是 $0$​，那么 $[l-1,r]$​ 和 $[l,r+1]$​ 必须是 $1$​，那么 $[l-1,r+1]$​ 就必须是 $0$​ 了。​若 $[l,r]$​ 是 $1$​，这说明 $[l+1,r]$​ 与 $[l,r-1]$​ 中至少有一个是 $0$​ ，那么 $[l+1,r+1]$​ 和 $[l-1,r-1]$​ 中也至少有一个是 $1$​，这又可以导出 $[l,r+1]$​ 和 $[l-1,r]$​ 中至少有一个是 $0$​，于是 $[l-1,r+1]$ 也是 $1$ 了。

有了这个性质，我们考虑按 $l+r$ 分组维护。特殊区间以及与它在两步之内的区间直接暴力计算。

现考虑一个询问 $[a,b]$，如果它所在的组内没有元素，则说明它的胜负只与 $a+b$ 的奇偶有关。若有元素，则找到比它小且最大的区间，按照之前的性质，$[a,b]$​​ 的胜负一定与这个区间有关。而以上操作都是可以用 map 维护的，总时间复杂度 $O(n\log n)$。

具体实现上，我们在暴力维护特殊区间相关区间时，是要按从小到大的顺序的，因为有可能相关区间之间会互相影响。具体处理手段可以看代码。

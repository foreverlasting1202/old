[1001 Miserable Faith]

solution

题面：一棵以 $1$ 为根的树，开始每个点颜色不同。定义一条边边权为 $1$ 当且仅当端点颜色不同，否则为 $0$。有四种操作：

  1. 将 $u$ 到 $1$ 的路径上的所有点都染成一个没出现过的颜色。
  2. 询问 $u$ 到 $v$ 的边权和。
  3. 询问 $u$​ 的子树内所有点到 $u$ 的边权和的和。
  4. 定义 $u$​​ 的最浅相同颜色的祖先到 $u$ 的距离为 $dis_u$，求 $\sum_{x=1}^n dis_x$ 。

做法：考场一眼就发现是 NOI 2021 D1T1 的改题了，第一个操作是 LCT 的 access ，第二个操作和第三个操作就是询问虚边。那我就维护实边，虚边就是总边长减重边了。第四个操作放在 LCT 上就是同求每一个点到它所在 splay 的最浅深度点的距离的和。冷静分析一下，由于第一种操作只会到 $1$，那我连 makeroot 都不用，直接 access ，然后在这过程中有个线段树维护子树边权和就好了。第四个操作只要你对 splay 的每个点的深度取个 $\min$，那这个就能拆成所有深度和减最小深度乘 splay 的点数，$\min(dep)$​ 只有在虚实边切换的时候会改变，那么都可以在 access 的时候顺手维护好。于是总复杂度就是 $O(n\log^2 n)$。

[1003 VC Is All You Need]

solution

题面：是否存在 $k$​ 维空间中的 $n$​ 个点和任意的黑白染色方案，使得都存在 $k-1$​ 维的超平面可以恰好把相同颜色的点分在同一边。

做法：看到样例，猜测一下就是 $[n\leq k+1]$​​。

证明的话我们先考虑二维，那随便选三个不共线的点肯定存在一条线，而选取四个点的话，你对角同色的情况就找不到一条线了。

然后归纳证明，第 $k$​​ 维的情况就考虑将其投影到一个 $k-1$​​ 维空间里同时满足有两个点投影重合，这样就可以证明了。大致就是这样了。

[1004 Another String]

solution

题面：定义两个相同长度的字符串的距离为相同位置不同字符的个数。现给出一个字符串 $S$ 和自然数 $k$，希望你对每一个 $i=1,…,n-1$​​，求出 $S[1,i]$ 与 $S[i+1,n]$ 两者有多少个子串对满足距离小于等于 $k$。

做法：不知道为什么他标算写得那么麻烦。

设 $f_{i,j}$ 表示 $S[i,j-1]$ 与 $S[j,n]$ 的 $\mathrm{lcp}$​，这里 $\mathrm{lcp}$ 定义为距离小于等于 $k$​​ 且最长的前缀。我们考虑 $f_{i,j}$ 对答案的贡献，设其为 $x$​，注意到它能影响的分割点只有 $i,…,j-1$，而分割点的前一段因为长度小于 $x$，那么得到的贡献就是它自身到 $i$ 的长度。后一段则都是 $x$。注意到 $p-i$ 可以拆成 $1\ast p-i$，那就可以维护两个差分数组，分别维护常数贡献和自身位置贡献（即记录自身位置用过几次），于是现在只要能求 $f_{i,j}$ 就行了。

原本想用后缀结构搞一搞的，冷静一下发现数据范围允许 $O(n^2)$ 了，那就维护个单调的指针，从后往前扫一遍就能求出 $f_{i,j}$ 了。总复杂度 $O(n^2)$。

不知道能不能做到 $O(n\log)$ 的级别。

[1005 Random Walk 2]

solution

题面：给出 $w_{i,j}$，从节点 $i$ 走到 $j$ 的概率为 $\frac{w_{i,j}}{\sum_{k=1}^n w_{i,k}}$，若从 $j$ 走到了 $j$ 就会停止。希望你对每一个 $(i,j)$ 求出，从 $i$ 出发，到 $j$​ 停止的​​概率。

做法：设 $f_{i,j}$ 表示从 $i$ 出发到 $j$ 停止的概率，容易写出转移式

发现是个线性方程组，但一算高斯消元的复杂度已经到了 $O(n^6)$，放弃。

冷静一下，发现 $f_{i,j}$​​​ 总是类似 $\sum a_{i,k}\cdot b_{k,j}$​​​​ 的物品，于是设 $f$​​ 构成的矩阵为 $F$​​，$w$​​ 构成的矩阵为 $W$​​，但 $W$​​ 去掉对角线，设 $w$​​ 的矩阵的对角线矩阵为 $W’$​​，容易发现 $W’+WF=F$​​，即 $F=(I-W)^{-1}W’$​​，于是写个求逆就好了。

时间复杂度 $O(n^3)$。

[1008 Supermarket]

solution

题面：给你 $m$​​​ 个 $01$​​ 串，等概率地选择其中任意个，然后或起来。设 $P(T|S)$​​ 表示或起来的串包含了 $S$​​ 的情况下，它包含 $T$​​ 的概率。求 $\sum_S\sum_T P(T|S)$​​​。

做法：注意到 $P(T|S)$​ 是条件概率，设取出包含 $S$​ 的概率是 $Q(S)$​​，那么 $P(T|S)=\frac{Q(S\cup T)}{Q(S)}$​，套上 $\sum$​ 就有

注意到 $Q(S)$​​ 和 $Q(S\cup T)$​​ 都是在做 $\sum_{S\And T=S} A(T)$​​ 的过程，而后者则多了 $2^{\mathrm{popcount}(S)}$​​ 的乘法因子（因为当 $S$​​ 非 $1$​​ 的位确定时，$S$​​ 含 $1$​​ 的这几位可以随意选取），于是做两次高维前缀和即可。

时间复杂度 $O(n2^n)$。

[1009 Array]

solution

题面：给出 $a_i$​，问有多少个区间满足区间内出现次数最多的数的个数大于其余数出现个数之和。

做法：不知道为什么我的做法这么复杂。

根据那个我忘记名字的投票法，我们枚举 $x$，将不是 $x$ 的数看成 $-1$，其余看成 $1$，附带着 $0$​ 号位做一遍前缀和，那每个点为终点的合法区间个数就是看它前面有几个数比它小。我们发现暴力维护这个的复杂度是 $O(nc)$，$c$​ 是颜色个数。然后冷静一下，我们对每个 $1$ 的位置寻求它和它连着的 $-1$ 的贡献。若我们维护一个以前缀和大小为下标的数组 $d_i$，那 $1$ 贡献就是个前缀和，而由于 $-1$​ 处的前缀和是连续的，那就是 $\sum_{j=L}^R \sum_{i=-N}^{j-1} d_i$，$L,R$ 分别为 $-1$ 处前缀和的下上界。这个东西显然可以分成两段，一部分是 $i\leq L-1$，这里还是维护 $d_i$ 的前缀和即可。另外一段的系数是等差数列，那我们就可以维护 $i\cdot d_i$​ 的前缀和。修改就是一个区间加，这些都是线段树可以做到的。时间复杂度 $O(n\log n)$。

另一思路不是维护 $d_i$，而是直接维护小于等于 $x$ 的前缀和个数 $s_x$。那查询就非常容易了，就是个区间和。修改操作的话，$1$ 处是简单的，就是区间加。而 $-1$​ 处则一样可以分成两段，一段是区间加，另一段是加一个等差数列，这照样是维护 $x\cdot s_x$​ 的前缀和可以解决的，还是线段树可以做到。时间复杂度仍然是 $O(n\log n)$。

出题人的 $O(n)$ 不太理解，大概是维护差分数组来延迟更新，从而不需要用线段树？不太懂哎，有没有懂哥教教我啊。

当然线段树都可以换成树状数组，线段树有点难以跑过去似乎。

[1011 Jsljgame]

solution

题面：有 $n$ 堆石子，每堆有 $a_i$ 个石子，先手的人不能取 $x$ 个石子，后手不能取 $y$ 个，不能取者输，问谁赢。

做法：特意去重温了超现实数，结果对这道题还是没有好的想法。

然后就只能大力猜结论再证了。

首先 $x=y$​ 是容易的，这可以直接转换为平等博弈，证明 $sg_i=i\mod x+\lfloor\frac{i}{2x}\rfloor x$​ 是容易的，归纳还是直接证都是可行的。

若 $x\neq y$ 就麻烦许多了。我们想用超现实数解，却发现大部分博弈都是 $x||0$​ 的情况，难以用超现实数表示。

于是就开始猜结论了。

如果所有数都小于 $x,y$，那就是普通的 $\mathrm{nim}$​ 游戏，全部异或即可。

否则 $a_n\geq \min(x,y)$​​。（不妨给 $a$ 排了序）

接下来我们可以归纳似的猜结论。

先考虑 $x<y$。​

若 $a_{n-1}<x$​​​​，​对于先手人而言，我们肯定想取 $a_n$​​ 一部分使得异或和为 $0$​​ 且 $a’_na_n-x$​​ 且 $\neq x$​​。否则必然会想办法不让后手人使总异或和为 $0$​​​​。

我们假设先手取之后是 $b_1,…b_n$​​​。我们换位思考后手人的操作再来取决先手人的操作。

若 $b_{n}<x$，则 $\oplus_{i=1}^n b_i=0$​ 就是先手人胜利的充要条件。

若 $b_n\geq x,b_{n-1}<x$​​，后手人肯定想使异或和为 $0$，那要求 $\oplus_{i=1}^n b_i\neq 0$，则必然存在 $\oplus_{i=1}^n b_i\oplus b_j\oplus (b_j-z)=0$，希望找到这样的 $j$，使得 $y\neq z$。若找到的 $j$ 只能使 $y=z$，那此时 $j$ 只能为 $n$（因为其它 $b_i<x$）。于是后手恶心人，取 $b_n$ $y-x$ 个石子，那先手只能取 $b_n$ $x$​ 个石子才能使异或和为 $0$​，可惜它取不了。先手人无法破解当前局面，因为后手人如果不能取 $y$ 个，那就一直重复取 $y-x$​​ 个的操作，先手人必败。

若 $\oplus_{i=1}^n b_i=0$​​。后手人肯定想要先手人只能取 $x$​​​ 个。注意到异或和为 $0$​​，这说明 $b_n$​​ 的最高位的 $1$​​ 在 $b_i$​​ 中出现了偶数次，同时也表明 $x$​​ 也有这样一个 $1$​​（因为 $b_{n-1}<x$​​）。那么 $0<b_j-b_j\oplus b_n\oplus (b_n-x)$​ 的解只有这些包含最高次 $1$​ 的数中才会出现。于是我们后手人每次找一个 $b_j$​，将其变成 $b_j\oplus b_n\oplus (b_n-x)$​。此时异或和的最高次 $1$​ 是存在的，先手人想要逆转局势，肯定要异或和重新变为 $0$​，那么动的还是包含最高次 $1$​ 的 $b_j$​。可惜，$b_j$​ 不包含 $b_n$​ 的情况下是奇数个，而我后手人是先动手的。那么我后手人只要一直重复上述操作，你先手人总会轮到必须把 $b_n$​ 变成 $b_n-x$​​​，那么你先手人必输无疑。所以这种情况还是先手人必输。

同理地，经过讨论，我们发现在 $x<y$​​ 的情况下，只有 $b_n<x$ 且 $\oplus_{i=1}^n b_i=0$ 的情况下先手才能赢。

而 $x>y$ 就等价于上述讨论了，于是先手必胜。

综上，$x=y$​ 时是个 $sg_i=i\mod x+\lfloor\frac{i}{2x}\rfloor x$​ 的平等博弈模型。

$x<y$​ 且 $a_n<x$​ 是 $\mathrm{nim}$​ 博弈模型。

$x<y,a_n\geq x$ 则看$a_n-x<a_n-\oplus_{i=1}^{n-1} a_i$ 且 $\neq x$ 是否成立。

$x<y$​​​ 的其他情况都是后手胜​。

$x>y$ 且 $a_n<y$ 是 $\mathrm{nim}$ 博弈模型。

$x>y$​ 且 $a_n\geq y$ 都是先手必胜。

[1012 Yet Another Matrix Problem]

solution

题面：给出 $n,m$，问对每个 $x\in [0,m]$，有多少个矩阵对 $(A,B)$，满足 $C=A*B,\sum c_{i,j}=x$，$A$ 是 $n\ast r$ 的矩阵，$B$ 是 $r\ast n$ 的矩阵，$r=n^m$，$a_{i,j},b_{i,j}\in[0,m]$。

做法：不得不说多项式板子间亦有差距！

先进行一波平静的推式子。

设 $A_k=p$​​ 的方案数为 $g_p$​​，隔板法可知 $g_p=\binom{p+n-1}{n-1}$​​。

再设 $A_kB_k=p$ 的方案数为 $f_p$，那 $f_p=(g\ast g)_p$，$\ast$ 为 $\mathrm{Dirichlet}$ 卷积，那么 $f$ 也就 $O(n\log n)$​ 求出了。$f_0$ 需特殊处理，直接考虑组合意义即可。

直接求 $\sum_{k=1}^r A_kB_k$ 不太好求，把它放在指数上，即设 $f$ 的 $\mathrm{OGF}$ 为 $F$，于是题目变成对每一个 $k\in [0,m]$，求

这个多项式快速幂就行了，时间复杂度 $O(n\log n)$。

坑：多项式取对数一般人的板子都是要求 $a_0=1$ 的，所以要先整体乘上逆元，求完之后再乘回去，注意乘回去的 $r$ 应该 $\mathrm{module}$ $998244352$。

[1013 Penguin Love Tour]

solution

题面：一棵树，有边权点权，每个点能让一条与自己相邻的边的边权减去自己的点权，问最小直径。

做法：先考虑贪心，然后不会。于是考虑二分答案。设 $f_{x,0/1}$​ 表示 $x$​​ 点用在它的孩子边还是父边的子树内的最长路径。转移就是考虑合并最大链次大链，有二分在就很容易判断了。时间复杂度 $O(n\log)$​。
